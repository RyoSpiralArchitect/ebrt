# Provisional Inspector design reference

`ebrt_inspector_provisional_concept_v0_4_1.png` is an implementation reference
for the temporary local Inspector only. It is not benchmark evidence and does
not freeze the eventual EBRT frontend, brand, navigation, or hosting model.

The concept establishes the open three-region layout, true-white canvas,
technical rule system, evidence-order rail, lineage comparison, and measured
usage dock. Runtime metadata, labels, evidence states, and values in the coded
viewer must always come from the normalized artifact; illustrative header
metadata inside the raster concept is not a source of truth.

`ebrt_inspector_provisional_implementation_v0_4_1.jpg` is the verified
1562×992 browser rendering over the committed incomplete four-arm artifact. It
records implementation fidelity and claim-boundary behavior, including the
visible `Cause decision: NOT READY` state. It is still a replaceable research
surface, not a product-design commitment.
