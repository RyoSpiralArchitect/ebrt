# EBRT

Event-driven Backward Reasoning for Test-Time Inference.

This default-branch snapshot preserves the frozen v0.1 mechanism harness. The
first pull request adds matched benchmarks, evidence, and the OpenAI Build Week
roadmap without modifying the monolith.

Current claim boundary: v0.1 demonstrates bounded, event-triggered backward
revision over structured reasoning states. It does not yet implement a trained
Transformer, autonomous semantic event detection, or GPT-5.6 integration.

```bash
python3 ebrt_monolith_v0_1.py --self-test
```
